# SF_Statistical_Mechanics

## Overview
Descriptive statistics and stochastic simulation of game mechanics. Focus on resource optimization and performance modeling in a multi-player environment. Results are presented to non-statisticians.

## Key Modules
### 1. Economic Value Modeling (VIP & Resource Packages)
Evaluation of the Return on Investment (ROI) of various in-game purchase options.
* **Methodology:** Normalization of currency value (Mushrooms per EUR) across heterogeneous package structures.
* **Goal:** Identification of the most cost-efficient acquisition paths for premium resources.

### 2. Longitudinal Guild Performance Tracking
A monitoring system for tracking group-level growth metrics over time.
* **Level Progression:** Weekly analysis of player level gains to identify growth plateaus.
* **Attribute Analysis:** Monthly tracking of primary statistics (Main Stats) to model the aggregate guild power level.

## Stakeholder Communication & Data Storytelling
A core objective of this project is the translation of statistical results for non-technical users. Insights are distilled into clear, visual reports.

## Tech Stack
* **Python:** Core logic and data processing.
* **Pandas:** Time-series handling for player progression.
* **Matplotlib/Seaborn:** Visualization of economic efficiency and growth trends.
